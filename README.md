# InfinityGuias

Portal de guias da InfinityGO para orientar clientes sobre o uso dos aplicativos e benefícios inclusos nos serviços da empresa.

O projeto centraliza tutoriais rápidos, com passo a passo, imagens e vídeos, para reduzir dúvidas recorrentes e facilitar o atendimento ao cliente.

## Conteúdo

- **Minha Infinity Go**: instalação, primeiro acesso, faturas, benefícios e desbloqueio de confiança.
- **ITTV Plus**: acesso pelo computador, Android e iOS, além do fluxo de criação/ativação de conta.
- **Imagina Só**: instalação e acesso pelo parceiro InfinityGO.
- **Teste de velocidade**: instruções para medir a conexão corretamente em PC, Android e iOS.

## Tecnologias

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React

## Como rodar

Instale as dependências:

```bash
npm install
```

Rode o ambiente de desenvolvimento:

```bash
npm run dev
```

Acesse:

```text
http://localhost:3000
```

## Estrutura principal

- `src/app`: páginas e rotas do site.
- `src/components`: componentes reutilizados na interface.
- `public/tutoriais`: imagens e vídeos usados nos tutoriais.
- `scripts`: scripts auxiliares do projeto.
